# AUTO_Si9000 自动阻抗计算
这是一个PCB阻抗正算、反算的Python项目，基于Si9000提供的dll方法

正算：
- 使用dll计算方法计算；
- 支持8种常见PCB叠构模型；


反算：
- 使用dll计算方法计算；
- 支持8种常见PCB叠构模型；
- 线宽、线距、对地距离调整方案为仅调整线宽大小，其它的距离会自动更新大小；
